Demonstration of waveform-compare — an xcorrSound tool

Waveform-compare compares two audio files and outputs the similarity. The tool can be used in quality assurance.

Choose a demo wave file, or upload your own








  • This tests xcorrsound functionality, but not performance which is affected by network bandwidth and server load.
  • The tool outputs "success" or "failure", depending on similarity for the best match (default threshold is 0.98), as well as offset and similarity (between 0 and 1) for best match, and in case of failure the block where the failure occurred.
  • For more on algorithm and output, see the xcorrSound web site.
