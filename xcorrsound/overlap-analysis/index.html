Demonstration of overlap-analysis — an xcorrSound tool

Overlap-analysis finds the overlap between the end of the first file and the beginning of the second file. It produces an offset and a value, the closer to 1 the value is the more reliable the match is.


Choose a demo wave file, or upload your own







  • This tests xcorrsound functionality, but not performance which is affected by network bandwidth and server load.
  • The tool outputs an offset in seconds and a match quality value. The closer to 1 the value is the more reliable the match is. The offset tells you how many seconds into the first audio file, the second audio file should be shifted to get the best match.
