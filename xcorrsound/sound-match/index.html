Demonstration of sound-match — an xcorrSound tool

Sound-match detects occurrences of a smaller audio file (e.g. a jingle) within a larger audio file or an index of audio files.

Choose a demo wave file, or upload your own







  • This tests xcorrsound functionality, but not performance which is affected by network bandwidth and server load.
